Chat App

A mobile chat application built with React Native and Expo. This app allows users to enter a chat room, customizable with a background color, and exchange messages.

Features

Start Screen: Users can enter their name and choose a background color for the chat screen.

Chat Screen: Displays the user's name in the navigation bar and applies the selected background color.

Navigation: Seamless transition between the start and chat screens using React Navigation.

Tech Stack

React Native

Expo

React Navigation

Installation & Running

Install dependencies:

npm install


Start the app:

npx expo start


Use the Expo Go app on your phone or an emulator to run the application.
